# 統合レポート作成アプリ

AFF・運用型広告のプラン／実績をアップロードし、以下を作成するStreamlitアプリです。

- 合算レポート
- 日別レポート
- 媒体別レポート
- Excelエクスポート

## 起動方法

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 主な仕様

- AFFプランと運用型プランはシートを選択可能
- 運用型実績の非表示シートを、非表示のまま読み込み
- 媒体別では運用型実績の9ローデータシートを横断し、A列の媒体詳細に対してG列のコスト、H列のCVを集計
- 月次ファイルで付与される `_[in_bank_数字]` などの末尾差分も吸収
- 媒体名称は媒体詳細の最初の `_` より前を自動取得
- Excel出力のフォントは Meiryo UI に統一
- レポート作成完了時に風船エフェクトを表示

`output_template.xlsx` は `app.py` と同じフォルダに置いてください。
